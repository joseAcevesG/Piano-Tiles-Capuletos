# Piano-Tiles-Capuletos

to set up the project, you need to create a virtual environment and install the requirements:

## Linux/MacOS

```bash
python3 -m venv .venv
source venv/bin/activate
pip install -r requirements.txt
```

## Windows

```bash
python -m venv .venv
venv\Scripts\activate
pip install -r requirements.txt
```
